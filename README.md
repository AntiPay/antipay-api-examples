# AntiPay API Examples

Official integration examples for AntiPay.

This repository contains sample code for common stacks.

## Examples
- Node.js
- Next.js
- PHP
- Laravel
- Postman collection

## Important
- Never put your API key in frontend code
- Always use backend calls for verification
- Webhook should point to your server endpoint

## Base URL
https://pay.antipay.site/v1/
